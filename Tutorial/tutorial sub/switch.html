C Switch Statement

The switch statement allows you to select one of many code blocks to be executed based on the value of an expression.

Basic Syntax

switch(expression) {
    case constant1:
        // code to be executed if expression equals constant1
        break;
    case constant2:
        // code to be executed if expression equals constant2
        break;
    // more case statements...
    default:
        // code to be executed if no case matches
}

Key Points:

  • The expression must evaluate to an integer or character
  • Case values must be constants (no variables)
  • The break statement exits the switch block
  • The default case is optional

Practical Example

// Program to display day of week
#include <stdio.h>

int main() {
    int day = 4;
    
    switch (day) {
        case 1:
            printf("Monday");
            break;
        case 2:
            printf("Tuesday");
            break;
        case 3:
            printf("Wednesday");
            break;
        case 4:
            printf("Thursday");
            break;
        case 5:
            printf("Friday");
            break;
        case 6:
            printf("Saturday");
            break;
        case 7:
            printf("Sunday");
            break;
        default:
            printf("Invalid day number");
    }
    
    return 0;
}

The break Keyword

When C reaches a break keyword, it breaks out of the switch block. Without it, execution will "fall through" to the next case. 

int x = 2;
switch (x) {
    case 1: printf("One");
    case 2: printf("Two");   // No break - fall through
    case 3: printf("Three");
}
// Output: "TwoThree"

This fall-through behavior can be useful in some cases, but is often a source of bugs if not intentional.

When to Use Switch vs If-Else

Switch If-Else
Tests equality of single variable against constant values Tests complex conditions with relational operators
More efficient for many discrete values More flexible for range checks
Works with integers and characters Works with all data types

Nested Switch Statements

int outer = 1, inner = 2;
switch(outer) {
    case 1:
        switch(inner) {
            case 1: printf("Inner 1"); break;
            case 2: printf("Inner 2"); break;
        }
        break;
    case 2:
        printf("Outer 2");
        break;
}
