C If...Else Statements

Conditional statements are used to perform different actions based on different conditions.

if Statement

if (condition) {
    // code to be executed if condition is true
}

The condition must evaluate to true (non-zero) or false (zero).

if...else Statement

if (condition) {
    // code if true
} else {
    // code if false
}

else if Statement

if (condition1) {
    // code if condition1 true
} else if (condition2) {
    // code if condition2 true
} else {
    // code if all false
}

Ternary Operator

variable = (condition) ? expressionTrue : expressionFalse;

This is a shorthand if...else that assigns a value based on a condition.

Practical Example

int time = 20;
if (time < 18) {
    printf("Good day.");
} else {
    printf("Good evening.");
}
// Outputs "Good evening."
