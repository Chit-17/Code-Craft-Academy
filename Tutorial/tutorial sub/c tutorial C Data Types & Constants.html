C Data Types & Constants

C has various data types to store different kinds of values, and constants to represent fixed values that cannot be changed.

Basic Data Types

Data Type Size (bytes) Format Specifier Range
int 4 %d -2,147,483,648 to 2,147,483,647
float 4 %f 1.2E-38 to 3.4E+38
double 8 %lf 1.7E-308 to 1.7E+308
char 1 %c -128 to 127 or 0 to 255

Type Modifiers

  • signed - Can store both positive and negative values (default)
  • unsigned - Can store only positive values (doubles positive range)
  • short - Uses less memory, smaller range
  • long - Uses more memory, larger range
Example 
#include <stdio.h>

int main() {
    // Different data types
    int integerVar = 42;
    float floatVar = 3.14;
    char charVar = 'A';
    
    // Type modifiers
    unsigned short age = 25;
    long long bigNumber = 123456789012345LL;
    
    // Printing with format specifiers
    printf("Integer: %d\n", integerVar);
    printf("Float: %.2f\n", floatVar);
    printf("Character: %c\n", charVar);
    
    return 0;
}

Constants

Constants are fixed values that cannot be altered during program execution.

Types of Constants

Literal Constants

  • Integer: 123, -456
  • Float: 3.14, -0.5
  • Character: 'A', '1'
  • String: "Hello"

Defined Constants

  • #define PI 3.14159
  • const float PI = 3.14159;
  • Enumeration: enum { RED, GREEN, BLUE };

The const Keyword

const int MAX_VALUE = 100;
const float PI = 3.14159;

Variables declared with const cannot be modified after initialization. Attempting to modify them will cause a compilation error.

Example 
#include <stdio.h>

// Macro constant
#define MAX_SIZE 100

int main() {
    // const variable
    const float PI = 3.14159;
    
    int radius = 5;
    float area = PI * radius * radius;
    
    printf("Area of circle: %.2f\n", area);
    printf("Max size: %d\n", MAX_SIZE);
    
    // PI = 3.14; // Error: assignment of read-only variable
    
    return 0;
}
